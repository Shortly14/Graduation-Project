# Graduation-Project
Graduation Project Code
OverviewWoundSense is a deep learning framework developed to perform binary classification of wound infections, distinguishing between infected and non-infected states using standard photographic images. Developed within the Department of Artificial Intelligence and Data Science, this system addresses the critical need for timely wound assessment in resource-limited and remote healthcare settings where expert clinical evaluation may be unavailable. It is designed to serve as a robust, automated first-line decision-support tool.  Key Findings & PerformanceOptimal Configuration: The best-performing model leverages a ConvNeXt-Tiny backbone paired with a soft spatial attention (Overlay) segmentation pipeline.  Peak Metrics: The optimal configuration achieved a test accuracy of 92.76%, an F1-score of 0.9276, and an AUC-ROC of 0.9858 on a held-out test set.  Context Preservation: The research demonstrates that the method of integrating segmentation is highly consequential. Traditional hard-cropping discards diagnostically relevant perilesional context (such as surrounding erythema), degrading performance. The novel overlay approach successfully guides the classifier's spatial attention while preserving the full image field.  MethodologyThe study systematically evaluates and benchmarks three distinct end-to-end classification pipelines:  Pipeline 1 (No Segmentation): Direct classification straight from raw, unsegmented wound photographs.  Pipeline 2 (Hard Crop): A pipeline where a lightweight U-Net model (LightUNet) localizes the wound region, and the tight bounding-box crop is subsequently classified.  Pipeline 3 (Overlay): A pipeline where a segmentation mask (from ImprovedUNet) is applied as a soft spatial attention signal, attenuating background pixels by 50% without discarding peripheral contextual information.  Each pipeline was tested against three state-of-the-art architectures: EfficientNet-B3, ResNet-50, and ConvNeXt-Tiny. The classification models were trained using a three-phase progressive fine-tuning strategy, beginning with frozen backbones and utilizing MixUp augmentation to safely adapt ImageNet-pretrained representations to the wound classification domain.  DatasetThe models were trained, validated, and tested on a curated, balanced dataset of 1,022 clinical wound photographs. The dataset features high-precision pixel-level wound region masks and expert binary infection labels, providing a rigorous foundation for both the segmentation and classification stages.
